Fitur Utama

File Upload & Validasi

  • Mendukung drag & drop file KMZ
  • Validasi format file (.kmz)
  • Batasan ukuran (max 50MB, min 1KB)
  • Validasi struktur internal
  • Pengecekan file KML dan placemarks

Export Settings

  • Format: Excel (.xlsx) / CSV (.csv)
  • Pemilihan kolom (Name, Latitude, Longitude)
  • Mode output (separate/merged files)
  • Kustomisasi nama file

Data Preview

  • Table View dengan fitur sort & filter
  • Map View dengan multiple layers
  • Marker clustering
  • Custom markers berdasarkan tipe
  • Tool pengukuran jarak & area

Spesifikasi Teknis

Dependencies

  • JSZip 3.7.1
  • XLSX 0.17.5
  • Leaflet
  • Bootstrap 5.3.3
  • Bootstrap Icons

Browser Support

  • Modern browsers
  • Responsive design
  • Drag & drop API
  • Modern JavaScript

Batasan Sistem

  • Max file: 50MB
  • Min file: 1KB
  • Input: KMZ only
  • Output: XLSX/CSV
